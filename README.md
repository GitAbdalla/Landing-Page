# Landing Page with Dark Mode

## Overview  
This is a simple landing page featuring a dark mode toggle for better user experience. The page is designed to be responsive and visually appealing in both light and dark themes.  

## Features  
✅ Dark mode toggle  
✅ Smooth transitions between themes  

## Technologies Used  
- **HTML** for structure  
- **CSS** for styling  
- **JavaScript** for dark mode functionality
